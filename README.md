# Digital Krishna

A spiritual AI companion powered by Gemini and the wisdom of the Bhagavad Gita.

## Directory Structure

- `digital_krishna/`: Flutter project (New implementation)
- `digital-krishna-react/`: React Native/Capacitor project (Original implementation)

## How to Run (Flutter)

1. Navigate to the project directory:
   ```bash
   cd digital_krishna
   ```

2. Get dependencies:
   ```bash
   flutter pub get
   ```

3. Run the app:
   ```bash
   flutter run
   ```

## Configuration

Ensure `.env` file exists in `digital_krishna/` with your `GEMINI_API_KEY`.

## Features

- **Chat with Krishna**: Ask questions and receive guidance based on the Bhagavad Gita.
- **Verse of the Day**: Daily shloka display.
- **Voice Output**: Text-to-speech support for divine responses.
- **Multilingual Support**: Supports English, Hindi, and Marathi context matching.
